# 更新日志

本项目版本遵循 `主.次.修订`，每次发布对应一个 GitHub Release（含免安装 exe）。

## v0.4.0

- **按时段拆分饮水目标**：全天目标拆成「上班前 / 上班时段 / 下班后」三段并显示各自毫升数，
  解决"目标是全天、却只在上班时段统计"的口径不一致；AI 按活跃时段分配，无 AI 时按比例兜底。
- **Scoop 一行安装**：新增 Scoop 清单，支持 `scoop install` 安装并 `scoop update` 升级，
  配置与喝水记录通过 persist 保留不丢。
- 新增 MIT `LICENSE`；README 把 Scoop 安装列为首选方式。

## v0.3.0

- **修 bug**：活跃/勿扰时段支持跨午夜（如 22:00–02:00）；忽略次数改为按天清零（不再越积越多）。
- **开机自启**：托盘新增「开机自启」开关；自启逻辑识别打包态（exe 直接自启）。
- **历史记录编辑**：网页可查看今日每条记录并逐条删除。
- **近 30 天趋势**：网页新增达标天数 / 日均 / 总量卡片。
- 健康提醒文案不再含具体毫升数字，避免与固定目标冲突。
- 加入版本号（托盘悬停与网页显示）。
- 新增单元测试（公式 / 时段 / 配置 / 存储，16 项）。

## v0.2.0

- **AI 结合健康状况定制方案**：可勾选常见健康状况（尿酸/痛风、低血糖、糖尿病等）并自由补充，
  AI 据此调整每日目标、给出节奏建议、分条提示与个性化提醒文案。
- **安全**：肾病 / 心衰 / 透析等需限水的情况提示遵医嘱、不擅自给危险数值，全程附免责声明；
  目标夹在 1200–4000ml。
- 每日方案持久化，当天只调一次 AI（重启不重算）。

## v0.1.0

首个发布版，核心功能成型：

- 系统托盘常驻 + 右下角悬浮小面板 + 本地网页控制面板（127.0.0.1:8765）。
- **自适应提醒**：喝完重新计时、忽略也会持续提醒，活跃/勿扰时段可配。
- 喝水打卡、撤销上一杯、AI 智能目标、AI 日报/周报（报告缓存 + 强制重生）。
- 可选天气接入（wttr / OpenWeather / 和风）让目标更精准。
- 报告推送：邮件 / Server酱 / PushPlus。
- 本地 SQLite 存储（多线程加锁），完全离线可用。
- **免安装 exe**：PyInstaller 打包配置；API Key 支持 `WATER_AI_API_KEY` 环境变量。
- **自动构建**：GitHub Actions 在打 tag 时自动编译 Windows exe 并发布到 Release。
